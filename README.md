# Lista de Vetores — Python

Repositório com exercícios práticos sobre **vetores (listas)** em Python, desenvolvidos como parte de estudos de programação.

---

## Estrutura do Repositório

| Arquivo | Descrição |
|---|---|
| `exercício1.py` | Vetor fixo com 6 inteiros: soma de posições específicas, modificação de elemento e exibição |
| `exercício2.py` | Leitura de 6 valores inteiros e exibição na tela |
| `exercício3.py` | Leitura de 10 números reais, cálculo do quadrado de cada elemento e impressão dos dois vetores |
| `exercício4.py` | Leitura de vetor com 8 posições e soma dos valores nas posições X e Y informadas pelo usuário |
| `exercicio5.py` | Leitura de vetor com 10 inteiros e contagem de quantos são pares |
| `exercicio6.py` | Leitura de vetor com 10 posições e exibição do maior e menor elemento |
| `exercicio7.py` | Leitura de 10 inteiros, exibição do vetor, do maior elemento e sua posição |
| `exercicio8.py` | Leitura das notas de 15 alunos e cálculo da média geral |
| `exercicio9.py` | Leitura de 10 números reais, contagem dos negativos e soma dos positivos |
| `exercicio10.py` | Leitura de 5 valores e exibição de todos junto com o maior, menor e média |
| `exercicio11.py` | Leitura de 5 valores e exibição das posições onde se encontram o maior e o menor |
 
---

## Conceitos Usados

- Declaração e inicialização de vetores
- Leitura de dados via `input()`
- Percorrimento com laços `for`
- Operações sobre elementos (soma, contagem, comparação)
- Exibição formatada de resultados

---

## Como Executar

Certifique-se de ter o **Python 3** instalado. Então, no terminal:

Bash:
```
python exercicio5.py
```
CMD:
```
py exercicio5.py
```

Substitua `exercicio5.py` pelo arquivo desejado.
